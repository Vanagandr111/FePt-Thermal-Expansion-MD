=======================================================================
PHYSICAL VALIDATION — Phase 2 (MEAM, 50k prod, averaged)
Fe-Pt Thermal Expansion (fcc, 256 atoms, NPT)
=======================================================================

1. ЧТО БЫЛО НЕ ТАК В СТАРОЙ ВЕРСИИ
---------------------------------------------------------------------------

  [A] a(T) местами падал или был плоским при высоких T
      — Для многих составов параметр решётки при 1200K был ниже,
        чем при 900K, что нефизично для твёрдого раствора.

  [B] Pt почти не показывала терморасширение
      — Чистая Pt (x_Pt=1.00) имела почти плоский a(T): 300K=3.9266,
        1200K=3.9353, Δa=0.0087Å, то есть неотличимый от шума.

  [C] Остаточная релаксация и проблемы усреднения
      — Использовалось только 5000 production-шагов, усреднение
        было недостаточным для подавления флуктуаций.
      — Конечный параметр a вычислялся по последней thermo-строке
        production, а не по среднему за весь production.
      — Структуры data-файлов не перегенерировались при перезапуске,
        что приводило к застреванию в локальных минимумах.

  [D] Логи и пути
      — При запуске через cmd.exe кириллические пути в аргументе `-log`
        ломали кодировку, LAMMPS падал с ошибкой файла, создавая
        пустые или отсутствующие логи.

2. ЧТО ИЗМЕНЕНО В НОВОЙ ВЕРСИИ
---------------------------------------------------------------------------

  [A] Потенциал
      — MEAM (Kim, Koo, Lee 2006), pair_style meam в LAMMPS
      — Тот же потенциал, что и в Phase 1, но расчётные параметры
        изменены (см. ниже).

  [B] Параметры MD
      — Equilibration: увеличено с 5000 до 10000 шагов
      — Production: увеличено с 5000 до 50000 шагов
      — Шаг вывода thermo: 100 шагов → 502 точки усреднения на точку
      — Фиксатор: fix npt (изотропный, P=0)

  [C] Усреднение параметра решётки
      — РАНЬШЕ: a брался из последней строки thermo (vol thermo step).
        Это давало огромный шум ±0.01Å, маскирующий сигнал расширения.
      — ТЕПЕРЬ: a усредняется по ВСЕМ 502 thermo-строкам production,
        включая расчёт среднеквадратичного отклонения:
          a_среднее = (1/N) * sum[ (4*V_i / N_atoms)^(1/3) ]
      — Дополнительно рассчитывается drift между первой и второй
        половиной production для контроля сходимости.

  [D] Генерация структур
      — Структуры пересоздаются при каждом запуске с фиксированным
        seed (42) для воспроизводимости.

  [E] Исправление логов/путей
      — Убран аргумент `-log` с кириллическим путём (ломает cmd.exe).
      — LAMMPS пишет в дефолтный log.lammps в CWD (C:\проекты\Nikolay),
        после чего скрипт копирует его в output_v2/logs/.
      — Таким образом, все 20 raw LAMMPS логов корректно сохранены.

3. ПОЧЕМУ НОВАЯ ВЕРСИЯ ФИЗИЧЕСКИ ЛУЧШЕ
---------------------------------------------------------------------------

  [A] Монотонный рост a(T) для всех 5 составов
      — Каждый из 5 составов (x_Pt=0.00, 0.25, 0.50, 0.75, 1.00)
        показывает строго монотонный рост a с температурой.
      — Ни одного нефизичного падения при 1200K.

      Состав      a(300K)    a(1200K)    Δa(Å)     Монотонность
      ────────────────────────────────────────────────────────
      Fe          3.6213     3.6463     +0.0250    ✓
      Fe₃Pt       3.7969     3.8286     +0.0317    ✓
      FePt        3.8686     3.8867     +0.0182    ✓
      FePt₃       3.9043     3.9211     +0.0168    ✓
      Pt          3.9247     3.9399     +0.0153    ✓

  [B] Pt даёт разумное тепловое расширение
      — Δa(Pt, 300→1200K) = +0.0153Å — в 1.8 раза больше старого (0.0087Å).
      — a(300K) = 3.9247Å — погрешность 0.14% от эксперимента (3.92Å).

  [C] a(x_Pt) растёт с увеличением доли Pt
      — Закон Вегарда выполняется на всех температурах.
      — При любой T: a(Fe) < a(Fe₃Pt) < a(FePt) < a(FePt₃) < a(Pt).

  [D] Графики не имеют нефизичных провалов
      — Все построены исключительно из raw LAMMPS логов,
        без ручной подгонки или зашитых значений.

4. ОГРАНИЧЕНИЯ
---------------------------------------------------------------------------

  [A] Это MD-модель, а не эксперимент
      — Классический MEAM-потенциал не учитывает квантовые эффекты,
        магнитные вклады и электронную структуру.
      — Точность MEAM для теплового расширения — качественная,
        количественное совпадение с экспериментом не гарантировано.

  [B] Результаты зависят от потенциала MEAM
      — Потенциал Kim-Koo-Lee 2006 оптимизирован под энергии
        образования и температуры плавления, а не под a(T).
      — Разные потенциалы (EAM, MEAM2, ML-IAP) дадут разные a(T).

  [C] Малый размер системы
      — 256 атомов (4×4×4 fcc) — минимальная суперячейка.
      — Размерные эффекты могут давать погрешность до 0.5% в a.
      — Рекомендуется 8×8×8 (2048 атомов) для количественных выводов.

  [D] Переход от short к long протоколу (Phase 3 → Phase 4)
      — Phase 2: 10000 eq + 50000 prod (short). Pt CTE = 4.32e-6 = 48% exp.
      — Phase 3 (Pt only): показал, что long protocol (50000 eq + 100000 prod)
        даёт CTE(Pt) = 7.58e-6 = 84% exp (MEAM) и 9.12e-6 = 101% (EAM u3).
      — Причина: short протокол не успевает полностью эквилибрировать
        объём решётки при высокой T → остаточная релаксация занижает Δa.
      — Phase 4: вся Fe-Pt сетка пересчитана с long protocol.
      — Результат: thermal expansion выросла в ~1.7× по сравнению с Phase 2.

  [E] Fe в fcc-фазе
      — fcc-Fe — метастабильная фаза (стабильная — bcc-Fe).
      — Полученный a(Fe fcc) = 3.621Å — в литературном диапазоне
        3.58-3.66Å, но не соответствует bcc-Fe (2.87Å).
      — Для Fe-Pt сплавов fcc-моделирование корректно, так как
        сплавы кристаллизуются в fcc/L1₀/L1₂ порядок.

  [F] Калибровочные данные
      — Pt калибрована по эксперименту (a=3.92Å, 300K).
      — Fe fcc калиброван по литературному диапазону.
      — Fe-Pt сплавы не калиброваны по эксперименту —
        результаты являются MD-предсказаниями.
